Python Webapp (AI-assisted)

Simple Flask app scaffolded with AI assistance.

Requirements
```
python -m venv venv
venv\Scripts\activate
pip install -r requirements.txt
```

Run
```
python app.py
```

Git / Push (example)
```
git init
git add .
git commit -m "Initial scaffold"
# create repo on GitHub and add remote, then:
git remote add origin https://github.com/<your-username>/ai-python-webapp.git
git branch -M main
git push -u origin main
```
